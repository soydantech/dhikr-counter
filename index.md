### Information Stored Locally on Your Device

All personal and religious content is stored **locally on your device** using a SQLite database. This includes:

- **Favorite Dhikrs**: Dhikrs you mark as favorites  
- **Custom Dhikrs**: Dhikrs you create yourself  
- **Dhikr Catalogs**: Custom catalogs you create  
- **App Settings**: Language, theme, notifications, daily targets, timer intervals  
- **Widget Configuration**: Home screen widget state and counter data  
- **Notification Rules**: Reminder schedules and preferences

### Location Information (Qibla Compass)

The App requests location permission **only** when you use the **Qibla Compass** feature.

Location is used to:
- Calculate the direction of the Kaaba (Qibla)
- Display an approximate address (via reverse geocoding)

**Location Privacy Details:**
- Location is used **only in real time**
- Location data is **not stored** on your device or our servers
- Location data is **not shared** with third parties
- You can deny location access and still use all other features
- Location access occurs **only while the Qibla Compass screen is active**

### Third-Party Location Service

#### OpenStreetMap Nominatim API

We use **OpenStreetMap Nominatim API** for reverse geocoding in the Qibla Compass feature.

- Receives latitude and longitude coordinates
- Returns an approximate address
- Does not permanently store location data

**Policies:**
